# Web UI for GitHub Intelligent Agent

React + Next.js 前端界面

## 技术栈

- **Next.js 14** - App Router
- **React 18** - UI 框架
- **Tailwind CSS** - 样式
- **Lucide Icons** - 图标
- **React Query** - 数据获取

## 运行

```bash
npm install
npm run dev
```

## 功能

- 💬 Chat - 与 Agent 对话
- 📋 Issues - Issue 管理
- 🔀 PRs - Pull Request 管理
- 🏁 Milestones - 里程碑管理
